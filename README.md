## 📖 使用指南

### 1️⃣ 数据大屏

点击顶部导航的 **"数据大屏"**，然后切换不同的分析模块：

```
销售趋势 → 用户分析 → 年龄分析 → 城市消费 → 商品销售 → RFM分析 → 商品推荐
```

#### 重点功能：RFM客户分析 ⭐

RFM 是最强大的客户价值分析工具：

- **R (Recency)**: 最近一次购买距今天数
- **F (Frequency)**: 购买频率
- **M (Monetary)**: 消费金额

系统自动将客户分为 8 类：

| 客户类型 | 特征 | 运营策略 |
|---------|------|---------|
| 重要价值客户 | 高R高F高M | VIP服务，专属优惠 |
| 重要发展客户 | 高R低F高M | 提升频次，组合促销 |
| 重要保持客户 | 低R高F高M | 防止流失，定期回访 |
| 重要挽留客户 | 低R低F高M | 激活沉睡，专属折扣 |

查看 **TOP20 高价值客户表格**，重点维护这些用户！

### 2️⃣ AI 智能分析

点击顶部导航的 **"AI 智能推荐"**，选择分析类型：

#### 📄 数据分析报告
- 一键生成全面的业务分析报告
- 包含用户画像、销售趋势、优化建议
- 适合周报、月报、管理层汇报

#### 👤 用户行为分析
1. 选择要分析的用户
2. 点击"生成分析报告"
3. 获得该用户的详细画像和推荐建议
4. 适合VIP客户深度运营

#### 💡 推荐优化建议
- 获取推荐系统优化策略
- 技术实现建议
- 预期效果评估
- 适合产品和技术团队

## 🎯 核心算法

### RFM 客户分群算法
```
1. 计算每个用户的 R、F、M 原始值
2. 对每个维度进行 1-5 分评分（五分位数法）
3. 根据 RFM 组合规则，分配到 8 个客户群
4. 计算每个群的统计数据和价值
```

### 协同过滤推荐算法
```
1. 获取目标用户的购买品类
2. 找到购买相同品类的相似用户
3. 统计相似用户购买的商品（目标用户未购买）
4. 按购买人数排序，返回 TOP6
```

### 关联规则挖掘
```
1. 按用户分组，收集购买商品集合
2. 计算所有商品两两组合的共现次数
3. 按共现次数排序，返回 TOP10 组合
```
