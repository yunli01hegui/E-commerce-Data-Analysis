# 电商数据分析平台 - 项目说明

## 🎯 项目概述

这是一个基于 React + TypeScript + Recharts + DeepSeek AI 的全功能电商数据分析平台，提供数据可视化大屏和AI智能推荐两大核心功能。

## 📊 核心功能

### 一、数据分析大屏

包含7个详细的数据分析模块：

#### 1. **销售趋势分析** 📈
- 关键指标卡片（总销售额、订单总量、平均客单价、月度增长率）
- 月度销售趋势图（面积图+折线图）
- 日销售趋势图（最近30天）

#### 2. **用户分析** 👥
- 男女用户概览卡片（总消费、客单价、订单数、购买件数）
- 性别消费对比柱状图
- 品类偏好雷达图
- 性别用户TOP5城市分布

#### 3. **年龄分析** 📊
- 5个年龄段概览（18-25、26-35、36-45、46-55、56-65岁）
- 年龄段消费对比图
- 年龄分布详情曲线（18-65岁）
- 年龄段品类消费堆叠图
- 智能洞察卡片

#### 4. **城市消费分析** 🏙️
- 城市分级统计（一线、新一线、其他）
- TOP10城市消费排行
- 城市消费力矩阵散点图
- 城市详细列表

#### 5. **商品销售分析** 📦
- 品类销售占比饼图
- 热销商品TOP15排行
- 商品销售数据统计

#### 6. **RFM客户分析** 🎯
- RFM模型说明（Recency、Frequency、Monetary）
- 8类客户自动分群
- 客户价值分布矩阵
- TOP20高价值客户表格
- 针对性运营策略建议

#### 7. **商品推荐系统** 🎁
- 个性化推荐（协同过滤）
- 组合推荐（关联规则）
- 热门商品推荐
- 新品推荐

### 二、AI智能分析

基于 DeepSeek 大模型的智能分析功能：

1. **自动生成数据分析报告**
   - 总体概况和关键指标
   - 用户画像分析
   - 商品销售分析
   - 优化建议

2. **用户消费行为分析**
   - 用户基本画像
   - 购买偏好分析
   - 用户价值评估
   - 个性化推荐建议

3. **推荐优化建议**
   - 推荐策略优化
   - 个性化推荐方案
   - 技术实现建议
   - 预期效果评估

## 🎨 核心算法

### 1. RFM客户分群算法

```typescript
R (Recency): 最近一次购买距今天数 → 1-5分（越小分数越高）
F (Frequency): 购买频率 → 1-5分（越高分数越高）
M (Monetary): 消费金额 → 1-5分（越高分数越高）

客户分群规则：
- 重要价值客户：R≥4, F≥4, M≥4
- 重要发展客户：R≥4, F≤2, M≥4
- 重要保持客户：R≤2, F≥4, M≥4
- 重要挽留客户：R≤2, F≤2, M≥4
- 一般客户：其他组合
```

### 2. 协同过滤推荐算法

```typescript
1. 获取目标用户购买的品类
2. 找到购买相同品类的相似用户
3. 获取相似用户购买的商品（目标用户未购买）
4. 按购买人数排序，返回TOP6
```

### 3. 关联规则挖掘

```typescript
1. 按用户分组，收集每个用户购买的商品
2. 计算所有商品两两组合的共现次数
3. 按共现次数排序，返回TOP10组合
```

## 🎯 业务洞察

### 客户分群运营策略

1. **重要价值客户**（高R高F高M）
   - 提供VIP专属服务
   - 优先购买权和新品推送
   - 建立长期关系

2. **重要发展客户**（高R低F高M）
   - 提高购买频次
   - 推送相关商品组合
   - 优惠券激励复购

3. **重要挽留客户**（低R低F高M）
   - 主动回访了解原因
   - 专属折扣和优惠
   - 重新激活沉睡客户

4. **一般客户**
   - 内容营销提升认知
   - 新人优惠降低门槛
   - 社交裂变扩大基数
